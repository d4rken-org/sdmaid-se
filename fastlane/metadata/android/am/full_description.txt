<i>SD Maid 2/SE</i> Android ዎን ንጹህና ስርዓታማ ለማድረግ የሚረዳ አስተማማኝ ረዳትዎ ነው።

ማንም ፍጹም አይደለም፣ Android ምም ፍጹም አይደለም።

* አስቀድሞ ያስወገዷቸው ብዙ ብዙ ፕሮግራሞች ነገር ይተዋሉ።
* ሎጎች፣ የስህተት ሪፖርቶች እና ሌሎች የማይፈልጓቸው ፋይሎች ያለማቋረጥ እየተፈጠሩ ናቸው።
* ማከማቻዎ የማያውቋቸው ፋይሎችና አቃፊዎችን እያሰባሰበ ነው።
* በጋለሪዎ ውስጥ ተባዥ ፎቶዎች።

እዚህ ላይ መቀጠሉ አያስፈልግም… SD Maid 2/SE ይርዳዎት!

SD Maid 2/SE የትኞቹ አፕሊኬሽኖች ፋይሎችን በእርስዎ መሣሪያ ላይ እንደፈጠሩ የሚያውቅ የፋይል ማስተዳደሪያ ነው። ፋይሎችን ከተጫኑ አፕሊኬሽኖች ጋር በማነጻጸር ማህደረ ትውስታን ደህንነቱ በተጠበቀ ሁኔታ ለማስለቀቅ ይረዳዎታል።

✨ ፕሮግራሞችን ካወረዱ በኋላ ማጽዳት

ፕሮግራሞች ፋይሎቻቸውን ከተወሰነላቸው አቃፊዎች ውጭ ከፈጠሩ ፕሮግራሞቹን ካወረዱ በኋላም ፋይሎቹ ሊቆዩ ይችላሉ። "CorpseFinder" መሣሪያ የፕሮግራም ቀሪዎችን ያገኛል፣ የትኛው ፕሮግራም ይዟቸው እንደነበር ይነግርዎታል እና ለመሰረዝ ይረዳዎታል።

🔍 መሣሪያዎን በብልሃት ፈልጉ

ባዶ አቃፊዎችን፣ ጊዜያዊ ፋይሎችን፣ አስቀድሞ የተጫኑ ፕሮግራሞችን እና ሌሎችን ያጣሩ። የራስዎን የፍለጋ መስፈርቶችም ሊፈጥሩ ይችላሉ። "SystemCleaner" መሣሪያ በተለያዩ መስፈርቶች ላይ በመመርኮዝ ፋይሎችን በራስ-ሰር ይፈልጋል እና ይሰርዛል።

🧹 ተጨማሪ ፋይሎችን እና የተደበቁ መሸጎጫዎችን ሰርዙ

ትናንሽ ምስሎች፣ የቆሻሻ ቅርጫቶች፣ ያለበይነ-መረብ ካሸዎች እና ተጨማሪ: ፕሮግራሞቹ ራሳቸው ካላጸዱ፣ ይህ ፕሮግራም ያጸዳቸዋል። "AppCleaner" መሣሪያ ተጨማሪ ፋይሎች ያሏቸው ፕሮግራሞችን ያገኛል።

📦 ሁሉንም ፕሮግራሞችዎን ያስተዳድሩ

በመሣሪያዎ ላይ የተጫኑ ሁሉንም ፕሮግራሞች ሙሉ ዝርዝር ያግኙ። ነቅቶ፣ ታግዶ፣ የተጠቃሚ ወይም የስርዓት ፕሮግራም: ምንም ፕሮግራም ከእርስዎ ሊደበቅ አይችልም። "AppControl" መሣሪያ አፕሊኬሽኖችዎን እንዲፈልጉ፣ እንዲደረድሩ እና እንዲቆጣጠሩ የሚያስችል የአፕ ማስተዳደሪያ ነው።

📊 ቦታዎን እየተጠቀመ ያለው ምንድን ነው

የማከማቻ አስተዳደር ከመሳሪያ ማከማቻ፣ ከSD ካሮዶች እና ከUSB መሳሪያወች ጋራ ያሉ መተግበሪያወን፣ ሜዲያወን፣ ⍝ርዓቱን እና ሌሎች ፋይሎችን ⍝ለሚኸዐን አስተዳደርን ወስብስብ ሊሆን ይችላል። "StorageAnalyzer" በመሣሪያዎ ላይ ቦታ እንዴት ጥቅም ላይ እንደሚውል የሚያሳይ የፋይል አስተዳዳሪ ሲሆን የማከማቻ አስተዳደርዎን ቀላል ያደርጋል።

📷 ተባዥ ውሂብ ይፈልጉ

ተባዥ ቅጅዎች፣ በማኅበራዊ ሚዲያ የተላኩ ፎቶዎች ወይም ተመሳሳይ ትዕይንቶች ፎቶዎች: ጊዜ እየሄደ ሲሄድ ቅጂዎች ሊከማቹ ይችላሉ። "Deduplicator" መሣሪያ ፍጹም ተመሳሳይ ወይም ተቀራራቢ ፋይሎችን ያገኛል እና ተጨማሪ ቅጂዎችን ለመሰረዝ ይረዳዎታል።

👆 በምዝዛት አጽዱ

አቃፊ ይምረጡ እና ፋይሎቹን አንድ በአንድ ቀላል ምዝዛቶችን በመጠቀም ለማቆየት፣ ለመሰረዝ ወይም ለመዝለል ይሂዱ። "Swiper" መሣሪያ ምንም ነገር ከመሰረዑ በፊት ሁሉንምን ውሳነዎች እንዲገምጎሞት ያስችላጅ፣ እና ክፍለ-ጊዜውን አስቀምጀ ቁየተው ሊቀጥሉ ይችላሉ።

🗄️ ፎቶዎቹን ሳይሰርዑ ቦታ ይቀንሱ

ፎቶዎች ብዙ ማከማቻ ሊወስዱ ይችላሉ፣ ነገር ግን ሁልጊዜ ማስወገድ አይፈልጉም። "Media Squeeze" መሣሪያ ምስሎችዎን ብቁል የሚነቀነቀ ጥራት ቅንብሮች ግግ በመጠቀም ፎቶዎችዎን ሰቅሮ ቦታ እንዲቈንቡ ይገዓል።

ይህ ፕሮግራም ማስታወቂያ የሌለው ነው። አንዳንድ ባህሪዎች ክፍያ ያለው ማሻሻያ ይፈልጋሉ።

SD Maid 2/SE የ SD Maid 1/Legacy ተተኪ ነው።
ለአዲሱ የ Android ስሪቶች የተሻሻለ እና ማጽዳት ላይ ያተኮረ።

ይህ ፕሮግራም አሰልቺ ድርጊቶችን ለማስወጣት AccessibilityService API ን የሚጠቀሙ አማራጭ ባህሪዎች አሉት።
AccessibilityService API ን በመጠቀም ይህ ፕሮግራም በርካታ ፕሮግራሞች ላይ ስራዎችን ለማከናወን ቁልፎችን ሊጫን ይችላል፣ ለምሳሌ። መሸጎጫዎችን ማጽዳት።
ይህ ፕሮግራም መረጃ ለመሰብሰብ AccessibilityService API ን አይጠቀምም።

SD Maid 2/SE የፋይል አስተዳዳሪ እና አጽዳቂ ፕሮግራም ነው።
