<i>SD Maid SE</i> adalah asisten tepercaya Android Anda, agar tetap bersih dan rapi.

Tiada yang sempurna dan itu termasuk Android.
* Aplikasi yang baru saja dihapus meninggalkan beberapa jejak.
* Log, laporan galat dan file-file lainnya yang Anda tidak benar-benar butuhkan masih terus dibuat secara terus-menerus.
* Penyimpanan anda mengumpulkan file-file dan direktori-direktori yang Anda tidak kenali.

SD Maid SE bebas iklan. Beberapa fitur membutuhkan peningkatan berbayar.

SD Maid 2/SE ialah penerus dari SD Maid 1/Lawas.
Dioptimalkan untuk Android versi terbaru dan difokuskan untuk membersihkan.

SD Maid SE mempunyai fitur opsional yang memanfaatkan API AccessibilityService untuk mengotomatisasi aksi yang membosankan.
Dengan menggunakan API AccessibilityService, SD Maid SE dapat mengeklik tombol-tombol untuk Anda untuk melakukan pekerjaan di beberapa aplikasi, seperti menghapus cache.
SD Maid SE tidak menggunakan API AccessibilityService untuk mengumpulkan informasi pribadi Anda.
