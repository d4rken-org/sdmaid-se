<i>SD Maid SE</i> è una fidata domestica per Android, per aiutarti a tenerlo pulito e in ordine.

Nessuno è perfetto, nemmeno Android.
* Le app che hai già rimosso lasciano dei residui.
* Vengono costantemente creati registri, rapporti sugli arresti anomali e altri file indesiderati.
* La memoria si riempie di file e directory che non riconosci.

Fermiamoci qui... Lasciati aiutare da SD Maid SE:

* Rimuove i dati delle app già disinstallate
* Trova le cache nascoste delle app
* Rimuove i file di sistema superflui

SD Maid SE non contiene pubblicità. Alcune funzioni richiedono un aggiornamento a pagamento.

SD Maid 2/SE è il successore di SD Maid 1/Legacy.
Ottimizzato per le versioni Android più recenti e incentrato sulla pulizia.

SD Maid SE fornisce funzioni opzionali che utilizzano l'API AccessibilityService per automatizzare le azioni più noiose.
Con l'API AccessibilityService, SD Maid SE può interagire con i pulsanti per eseguire operazioni su più app, come l'eliminazione della cache.
SD Maid SE non utilizza l'API AccessibilityService per raccogliere informazioni.
