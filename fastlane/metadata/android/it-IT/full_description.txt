<i>SD Maid SE</i> è il vero Assistente per tenere pulito e ordinato il tuo dispositivo Android.

Nessuno è perfetto, nemmeno Android.
* Le app che disinstalli lasciano alcuni residui.
* Vengono continuamente creati file di Log, report sugli arresti anomali e altri file indesiderati.
* La memoria accumula file e directory che non riconosci.

Non andare oltre… Lascia che SD Maid SE ti aiuti:

* Rimuove i residui dei dati delle app già disinstallate
* Trova le cache nascoste delle app
* Rimuove i file superflui di sistema

SD Maid SE non contiene pubblicità. Alcune funzionalità richiedono un aggiornamento a pagamento.

SD Maid 2/SE è la nuova versione di SD Maid 1/Legacy.
Ottimizzato per le versioni più recenti di Android e incentrato sulla pulizia.

SD Maid SE fornisce funzioni opzionali che utilizzano l'API AccessibilityService per automatizzare le azioni più noiose.
Con l'API AccessibilityService, SD Maid SE può interagire con i pulsanti per eseguire operazioni su più app, come l'eliminazione della cache.
SD Maid SE non utilizza l'API AccessibilityService per raccogliere informazioni.
