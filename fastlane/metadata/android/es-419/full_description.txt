<i>SD Maid SE</i> es el asistente de confianza de tu Android, para mantenerlo limpio y ordenado.

Nadie es perfecto y Android tampoco lo es.
* Las aplicaciones que ya has eliminado dejan algo atrás.
* Los registros, informes de fallos y otros archivos que realmente no deseas se están creando constantemente.
* Tu almacenamiento está acumulando archivos y directorios que no reconoces.

No sigamos aquí... Deja que SD Maid SE te ayude:

* Limpia los datos de aplicaciones ya desinstaladas
* Eliminar archivos innecesarios y cachés ocultos
* Eliminar archivos de sistema no utilizados
* Crea tus propios filtros para archivos aleatorios
* Buscar y eliminar datos duplicados
* Analiza tu almacenamiento y elimina archivos grandes que ocupan espacio

SD Maid SE no tiene anuncios. Algunas funciones requieren una actualización de pago.

SD Maid 2/SE es el sucesor de SD Maid 1/Legacy.
Optimizado para las versiones más recientes de Android y centrado en la limpieza.

SD Maid SE cuenta con características opcionales que utilizan la API de AccessibilityService para automatizar acciones tediosas.
Utilizando la API de AccessibilityService, SD Maid SE puede hacer clic en botones por ti para realizar operaciones en varias aplicaciones, por ejemplo. borrando cachés.
SD Maid SE no utiliza la API de AccessibilityService para recopilar información.
