<i>SD Maid SE</i> είναι μια πραγματική νοικοκυρά για το Android, για να το κρατά καθαρό και περιποιημένο.

Κάνεις δεν είναι τέλειος και ούτε και το Android.
* Εφαρμογές που έχετε ήδη αφαιρέσει πάντα αφήνουν κάτι πίσω.
* Αρχεία καταγραφής και διάφορα άλλα αρχεία που δεν θέλετε αυτά δημιουργούνται διαρκώς.
* Η εσωτερική μνήμη συλλέγει αρχεία και καταλόγους που δεν γνωρίζετε.

Ας μην συνεχίσουμε εδώ... Αφήστε το SD Maid SE να σας βοηθήσει:

* Καθαρίστε αρχεία απο ήδη απεγκατεστημενες εφαρμογές
* Βρειτε αρχεία κρυφής μνήμης από εφαρμογές
* Αφαιρέστε τα περιττά αρχεία συστήματος

Το SD Maid SE είναι χωρίς διαφημίσεις. Ορισμένες λειτουργίες απαιτούν πληρωμένη αναβάθμιση.

Το SD Maid 2/SE είναι ο διάδοχος του SD Maid 1/Legacy.
Βελτιστοποιημένο για νεότερες εκδόσεις Android και επικεντρομενο στον καθαρισμό.

Το SD Maid SE διαθέτει προαιρετικές δυνατότητες που χρησιμοποιούν το AccessibilityService API για την αυτοματοποίηση κουραστικών ενεργειών.
Χρησιμοποιώντας το AccessibilityService API, το SD Maid SE μπορεί να κάνει κλικ σε κουμπιά για να εκτελέσετε λειτουργίες σε πολλές εφαρμογές, π.χ. διαγραφή κρυφών μνημών.
Το SD Maid SE δεν χρησιμοποιεί το AccessibilityService API για τη συλλογή πληροφοριών.
