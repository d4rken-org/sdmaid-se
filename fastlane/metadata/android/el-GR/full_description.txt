Το <i>SD Maid SE</i> είναι η αξιόπιστη βοήθεια του Android σας, για να διατηρείται καθαρό και τακτοποιημένο.

Κανείς δεν είναι τέλειος, ούτε και το Android.
* Εφαρμογές που έχετε ήδη αφαιρέσει αφήνουν κάτι πίσω.
* Αρχεία καταγραφής και διάφορα άλλα αρχεία που δε θέλετε δημιουργούνται διαρκώς.
* Ο αποθηκευτικός χώρος σας συλλέγει αρχεία και καταλόγους που δε γνωρίζετε.

Ας μη συνεχίσουμε εδώ… Αφήστε το SD Maid SE να σας βοηθήσει:

* Καθαρίστε αρχεία από ήδη απεγκατεστημένες εφαρμογές
* Βρειτε αρχεία κρυφής μνήμης από εφαρμογές
* Αφαιρέστε τα περιττά αρχεία συστήματος

Το SD Maid SE δεν περιέχει διαφημίσεις. Ορισμένες λειτουργίες απαιτούν αναβάθμιση επί πληρωμή.

Το SD Maid 2/SE είναι ο διάδοχος του SD Maid 1/Legacy.
Βελτιστοποιημένο για νεότερες εκδόσεις Android και επικεντρωμένο στον καθαρισμό.

Το SD Maid SE διαθέτει προαιρετικές δυνατότητες που χρησιμοποιούν το AccessibilityService API για την αυτοματοποίηση κουραστικών ενεργειών.
Χρησιμοποιώντας το AccessibilityService API, το SD Maid SE μπορεί να πατήσει κουμπιά για να εκτελέσετε λειτουργίες σε πολλές εφαρμογές, π.χ. διαγραφή κρυφών μνημών.
Το SD Maid SE δε χρησιμοποιεί το AccessibilityService API για τη συλλογή πληροφοριών.
