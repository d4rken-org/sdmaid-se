<i>SD Maid SE</i> is your Android’s trusted assistant, to keep it clean and tidy.

אף אחד לא מושלם וגם אנדרואיד לא.
* אפליקציות שכבר הסרת משאירות שאריות.
* יומנים, דוחות קריסה וקבצים אחרים שאתה לא באמת רוצה נוצרים כל הזמן.
* זיכרון המכשיר שלך אוסף קבצים וספריות שאינך מזהה.

Let’s not go on here… Let SD Maid SE help you:

* נקה נתונים מאפליקציות שכבר הוסרו
* מצא מטמונים של אפליקציות נסתרות
* הסר קבצי מערכת מיותרים

SD Maid SE היא ללא פרסומות. חלק מהפונקציות דורשות שדרוג בתשלום.

SD Maid 2/SE הוא היורש של SD Maid 1.
מותאם לגרסאות אנדרואיד חדשות יותר ומתמקד בניקוי.

SD Maid SE בעלת תכונות אופציונליות המשתמשות ב-AccessibilityService API כדי להפוך פעולות מייגעות לאוטומטיות.
באמצעות ה-API של AccessibilityService, SD Maid SE יכולה ללחוץ על כפתורים כדי שתוכל לבצע פעולות באפליקציות מרובות. מחיקת מטמונים.
SD Maid SE לא משתמשת ב-AccessibilityService API כדי לאסוף מידע.
