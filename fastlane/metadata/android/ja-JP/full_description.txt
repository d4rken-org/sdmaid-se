<i>SD Maid 2/SE</i> はあなたのAndroidの信頼できるアシスタントであり、清潔できれいに保ちます。

完璧な人がいないように、Androidも同様です。
* 既に削除したアプリが何かを残していることがあります。
* ログ、クラッシュレポート、その他のファイルは常に作成されます。
* ストレージは、認識できないファイルとフォルダを収集しています。
* ギャラリー内の写真を複製します。

これ以上はやめて… SD Maid SEにお任せください！

SD Maid 2/SEは、どのアプリがあなたのデバイス上に特定のファイルを作成したかを知ることに特化したアプリとファイルマネージャーです。 SD Maid 2/SEは、デバイスを検索し、インストールされているアプリとファイルを比較して、ストレージスペースを安全に確保するためのオプションを提供します。

✨ アプリのアンインストール後のクリーンアップ
アプリが指定されたフォルダ外にファイルを作成すると、アプリをアンインストールした後もファイルが残ります。 残留ファイル検索は、アプリの残留を見つけ、どのアプリに属していたかを教えてくれ、それらを削除するのに役立ちます。

🔍 スマートな方法でデバイスを検索
空のフォルダ、一時的なファイル、すでにインストールされているアプリなどをフィルタします。 独自の検索条件を作成することもできます。 システムクリーナーを使用すると、自動的にデバイスを検索し、さまざまな条件に基づいてファイルを削除することができます。

🧹 消耗ファイルと隠しキャッシュを削除する
サムネイル、ゴミ箱、オフラインキャッシュなど：このアプリがクリーンアップします。 アプリクリーナーは消耗可能なファイルを持つアプリを検索します。

📦 すべてのアプリを管理
お使いのデバイスにインストールされているすべてのアプリの包括的なリストを取得します。 有効、無効、ユーザーまたはシステムアプリ：アプリを非表示にすることはできません。 アプリ管理は、アプリの検索、並べ替え、制御を行うアプリマネージャーです。

📊 あなたのスペースをすべて使っているもの
ストレージ管理は、携帯電話のストレージ、SDカード、およびUSBデバイス全体のアプリ、メディア、システム、およびその他のファイルで複雑になる可能性があります。 ストレージ解析は、デバイスでの使用方法を表示するファイルマネージャーで、ストレージ管理を簡素化します。

📷 重複データを検索
重複したダウンロード、ソーシャルメディアまたは同じシーンのちょうど類似の写真を介して送信された写真：時間の経過とともにコピーが蓄積されることがあります。 重複ファイルは、まったく同じまたは同様のファイルを検索し、追加のコピーを削除するのに役立ちます。

このアプリは広告なしです。 いくつかの機能を利用するためにはアップグレードを購入する必要があります。

SD Maid 2/SEはSD Maid 1/Legacyの後継です。
新しいAndroid用に最適化されクリーンアップに焦点を当てています。

このアプリには、AccessibilityService API を利用して面倒な操作を自動化するオプション機能があります。
AccessibilityService API を使用すると、このアプリはボタンをクリックして複数のアプリ間で操作を行うことができます。 キャッシュを削除することができます。
このアプリは、AccessibilityService API を使用して情報を収集しません。

SD Maid 2/SE はファイルマネージャ&クリーナーアプリです。
