<i>SD Maid SE</i> is your Android’s trusted assistant, to keep it clean and tidy.

Niemand is perfect en Android ook niet.
* Apps die je al hebt verwijderd, laten iets achter.
* Er worden voortdurend logboeken, crashrapporten en andere bestanden gemaakt die je eigenlijk niet echt wilt hebben.
* Je opslagruimte verzamelt bestanden en mappen die je niet herkent.

Let’s not go on here… Let SD Maid SE help you:

* Ruim gegevens op van reeds verwijderde apps
* Vind verborgen app-caches
* Verwijder overbodige systeembestanden

SD Maid SE is advertentievrij. Voor sommige functies is een betaalde upgrade vereist.

SD Maid 2/SE is de opvolger van SD Maid 1/Legacy.
Geoptimaliseerd voor nieuwere Android-versies en gericht op opschonen.

SD Maid SE heeft optionele functies die gebruikmaken van de AccessibilityService API om vervelende handelingen te automatiseren.
Met behulp van de AccessibilityService API kan SD Maid SE op knoppen klikken om bewerkingen uit te voeren op meerdere apps, bijvoorbeeld. caches verwijderen.
SD Maid SE gebruikt de AccessibilityService API niet om informatie te verzamelen.
