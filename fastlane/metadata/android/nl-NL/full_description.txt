SD Maid 2/SE is de opvolger van SD Maid 1/Legacy.
Geoptimaliseerd voor nieuwere Android-versies en gericht op opschonen.

Deze app heeft optionele functies die gebruik maken van de Toegankelijkheidsservice API om vervelende acties te automatiseren.
Met behulp van de AccessibilityService API kan deze app op knoppen klikken waarmee u bewerkingen in meerdere apps kunt uitvoeren, bijv. caches aan het verwijderen.
Deze app maakt geen gebruik van de Toegankelijkheidsservice API om informatie te verzamelen.

SD Maid 2/SE is een bestandsbeheerder- en opschoonapp.
