<i>SD Maid SE</i>คือแม่บ้านที่คุณวางใจให้ดูแลอุปกรณ์แอนดรอยด์ของคุณให้สะอาดเรียบร้อยอยู่เสมอ

ไม่มีใครสมบูรณ์แบบ แอนดรอยด์ก็เช่นกัน
* แอพที่คุณลบออกไปแล้วได้ทิ้งบางอย่างเอาไว้
* บันทึกการทำงาน, รายงานข้อผิดพลาด และไฟล์ต่างๆ ที่คุณไม่ต้องการถูกสร้างขึ้นตลอดเวลา
* พื้นที่ในอุปกรณ์ของคุณเต็มไปด้วยไฟล์และไดเรกทอรี่ที่คุณไม่รู้จัก

อย่าปล่อยให้เป็นแบบนี้ต่อไป... ให้ SD Maid SE ช่วยคุณ:

* ลบเศษซากของแอพที่คุณได้ลบทิ้งไปแล้ว
* ค้นหาไฟล์แคชที่ซ่อนอยู่
* ลบไฟล์ไม่จำเป็นที่ระบบสร้างขึ้น

SD Maid SE ไม่มีโฆษณา และสามารถอัพเกรดเพื่อใช้ฟีเจอร์บางอย่าง

SD Maid 2/SE เป็นรุ่นต่อยอดความสำเร็จจาก SD Maid 1
ปรับปรุงเพื่อแอนดรอยด์รุ่นใหม่และเน้นไปที่การทำความสะอาด

SD Maid SE มีฟีเจอร์เพิ่มเติมที่ใช้ AccessibilityService API เพื่อช่วยจัดการงานน่าเบื่ออย่างอัตโนมัติ
เช่น ด้วยการใช้ AccessibilityService API ทำให้ SD Maid SE สามารถคลิกปุ่มแทนคุณเพื่อดำเนินการกับหลายๆ แอพในคราวเดียว การลบไฟล์แคช
SD Maid SE ไม่ได้ใช้ AccessibilityService API เพื่อเก็บรวบรวมข้อมูลของคุณ
