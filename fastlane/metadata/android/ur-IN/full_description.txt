<i>SD Maid 2/SE</i> آپ کے Android کا معتبر معاون ہے، اسے صاف اور منظم رکھنے کے لیے۔

کوئی بھی کامل نہیں ہے اور Android بھی نہیں ہے۔

* جو ایپس آپ نے ہٹا دی ہیں وہ کچھ نہ کچھ پیچھے چھوڑ جاتی ہیں۔
* لاگ، کریش رپورٹس اور دیگر فائلیں جو آپ نہیں چاہتے مسلسل بنائی جا رہی ہیں۔
* آپ کی اسٹوریج میں ایسی فائلوں اور فولڈرز جمع ہو رہے ہیں جن کو آپ نہیں سمجھتے۔
* آپ کی گیلری میں ڈپلیکیٹ تصویریں۔

آئیے یہاں آگے نہ بڑھیں… SD Maid 2/SE کو آپ کی مدد کرنے دیں!

SD Maid 2/SE ایک ایپ اور فائل مینیجر ہے جو جانتا ہے کہ آپ کے ڈیوائس پر کون سی ایپس نے فائلیں بنائیں۔ یہ نصب شدہ ایپس کے ساتھ فائلوں کا موازنہ کرتا ہے تاکہ آپ محفوظ طریقے سے اسٹوریج خالی کر سکیں۔

✨ ایپس ہٹانے کے بعد صفائی کریں

اگر ایپس اپنے مقررہ فولڈرز کے باہر فائلیں بناتی ہیں تو ایپس کو ان انسٹال کرنے کے بعد بھی فائلیں باقی رہ سکتی ہیں۔ "CorpseFinder" ٹول ایپ کے باقیماندہ حصوں کو تلاش کرتا ہے، بتاتا ہے کہ وہ کس ایپ سے تعلق رکھتے ہیں اور آپ کو ان کو حذف کرنے میں مدد دیتا ہے۔

🔍 اپنے ڈیوائس میں ذہین انداز میں تلاش کریں

خالی فولڈرز، عارضی فائلوں، پہلے سے انسٹال شدہ ایپس اور مزید کے لیے فلٹر کریں۔ آپ اپنی خود کی تلاش کی شرائط بھی بنا سکتے ہیں۔ "SystemCleaner" ٹول خودکار طور پر مختلف معیار کی بنیاد پر فائلیں تلاش اور حذف کرتا ہے۔

🧹 غیر ضروری فائلوں اور چھپے ہوئے کیش کو حذف کریں

تھمبنیلز، ٹریش بن، آف لائن کیش اور بہت کچھ: اگر ایپس اپنی صفائی نہیں کرتی ہیں، تو یہ ایپ کرے گی۔ "AppCleaner" ٹول غیر ضروری فائلوں والی ایپس تلاش کرتا ہے۔

📦 اپنی تمام ایپس کو منیج کریں

اپنے ڈیوائس پر نصب تمام ایپس کی جامع فہرست حاصل کریں۔ فعال، غیر فعال، صارف یا سسٹم ایپ: کوئی بھی ایپ آپ سے چھپ نہیں سکتی۔ "AppControl" ٹول ایک ایپ مینیجر ہے جو آپ کو اپنی ایپس تلاش، ترتیب اور کنٹرول کرنے دیتا ہے۔

📊 آپ کی تمام جگہ کو کیا استعمال کر رہا ہے

اسٹوریج مینجمنٹ پیچیدہ ہو سکتی ہے، ڈیوائس اسٹوریج، ایس ڈی کارڈز اور یو ایس بی ڈیوائسز میں ایپس، میڈیا، سسٹم اور دیگر فائلوں کے ساتھ۔ "StorageAnalyzer" ایک فائل منیجر ہے جو آپ کے ڈیوائس پر اسٹوریج کا استعمال دکھاتا ہے، آپ کی اسٹوریج مینجمنٹ کو سہل بناتے ہوئے۔

📷 نقل شدہ ڈیٹا تلاش کریں

نقل شدہ ڈاؤن لوڈز، سوشل میڈیا کے ذریعے بھیجی گئی تصاویر یا ایک جیسے منظر کی سمیلی تصاویر: وقت کے ساتھ کاپیاں جمع ہو سکتی ہیں۔ "Deduplicator" ٹول ایسی فائلوں کو تلاش کرتا ہے جو بالکل ایک جیسی ہوں یا سمیلی ہوں اور آپ کو اضافی کاپیوں کو حذف کرنے میں مدد دیتا ہے۔

👆 ایک سوائپ سے صفائی کریں

ایک فولڈر منتخب کریں اور آسان سوائپس سے فائلیں ایک ایک کر کے دیکھیں — رکھیں، حذف کریں، یا چھوڑ دیں۔ "Swiper" ٹول آپ کو کچھ بھی حذف ہونے سے پہلے تمام فیصلوں کا جائزہ لینے دیتا ہے، اور آپ سیشن بچا کر بعد میں جاری رکھ سکتے ہیں۔

🗜️ تصاویر حذف کیے بغیر جگہ بچائیں

تصاویر بہت زیادہ اسٹوریج لے سکتی ہیں، لیکن آپ ہمیشہ انہیں ہٹانا نہیں چاہتے۔ "Media Squeeze" ٹول آپ کی تصاویر کو قابل ترمیم معیار کی ترتیبات کے ساتھ کمپریس کرتا ہے تاکہ تصویریں محفوظ رکھتے ہوئے جگہ خالی کی جا سکے۔

یہ ایپ اشتہارات سے پاک ہے۔ کچھ خصوصیات کے لیے ادائیگی کی ضرورت ہے۔

SD Maid 2/SE، SD Maid 1/Legacy کا جانشین ہے۔
نئے Android ورژنوں کے لیے بہتر بنایا گیا اور صفائی پر مرکوز۔

اس ایپ میں اختیاری خصوصیات ہیں جو AccessibilityService API کو استعمال کرتے ہوئے مشکل کام کو خودکار بناتی ہیں۔
AccessibilityService API استعمال کرتے ہوئے، یہ ایپ آپ کے لیے بٹن دبا سکتی ہے تاکہ کئی ایپوں پر کارروائیاں انجام دی جائیں، مثال کے طور پر کیش کو حذف کرنا۔
یہ ایپ معلومات جمع کرنے کے لیے AccessibilityService API استعمال نہیں کرتی۔

SD Maid 2/SE ایک فائل منیجر اور کلینر ایپ ہے۔
