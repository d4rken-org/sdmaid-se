<i>SD Maid SE</i> - надзейны памочнік вашай АС Android, які падтрымлівае яе ў чысціні і парадку.

Ніхто не дасканалы і Android таксама.
* Праграмы, якія вы ўжо выдалілі, пакідаюць рэшткі пасля сябе.
* Увесь час ствараюцца журналы, справаздачы пра збоі і іншыя файлы, якія вам не патрэбныя.
* Ваша памяць назапашвае файлы і папкі, якія Вы не пазнаеце.

Давайце не марудзіць... Дазвольце SD Maid SE дапамагчы вам:

* Ачысціць даныя ужо выдаленых праграм
* Выдаліце файлы працяглага карыстання і схаваны кэш
* Выдаліце невыкарыстоўваныя сістэмныя файлы
* Стварайце ўласныя фільтры для розных файлаў
* Знайдзіце і выдаліце дублікаты дадзеных
* Прааналізуйце памяць і выдаліце вялікія файлы, якія займаюць месца

SD Maid SE не змяшчае рэкламы. Некаторыя функцыі патрабуюць платнага абнаўлення.

SD Maid 2/SE з'яўляецца пераемнікам SD Maid 1/Legacy.
Праграма аптымізавана для новых версій Android і сканцэнтравана на ачыстцы.

SD Maid SE мае дадатковыя функцыі, якія для аўтаматызацыі дзеянняў выкарыстоўваюць API службы спецыяльных магчымасцей.
Выкарыстоўваючы API службы спецыяльных магчымасцей, SD Maid SE можа націскаць кнопкі, каб вы маглі выконваць аперацыі ў некалькіх праграмах. выдаленне кэшу.
SD Maid SE не выкарыстоўвае API службы спецыяльных магчымасцей для збору інфармацыі.
