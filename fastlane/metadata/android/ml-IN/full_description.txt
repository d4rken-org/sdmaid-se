<i>SD Maid 2/SE</i> നിങ്ങളുടെ Android-ന്റെ വിശ്വസ്ത സഹായിയാണ്, വൃത്തിയായും ചിട്ടയായും സൂക്ഷിക്കാൻ.

ആരും തികഞ്ഞവരല്ല, Android-ഉം ഇല്ല.

* നീക്കംചെയ്ത ആപ്പുകൾ ഫയലുകൾ അവശേഷിപ്പിക്കുന്നു.
* ലോഗുകൾ, ക്രാഷ് റിപ്പോർട്ടുകൾ, അനാവശ്യ ഫയലുകൾ നിരന്തരം സൃഷ്ടിക്കപ്പെടുന്നു.
* അജ്ഞാത ഫയലുകളും ഫോൾഡറുകളും സ്റ്റോറേജിൽ ശേഖരിക്കുകയാണ്.
* ഗ്യാലറിയിൽ ഡ്യൂപ്ലിക്കേറ്റ് ഫോട്ടോകൾ.

SD Maid 2/SE സഹായിക്കട്ടെ!

SD Maid 2/SE ഒരു ആപ്പ്, ഫയൽ മാനേജരാണ്, ഏതൊക്കെ ആപ്പ് ഏതൊക്കെ ഫയൽ സൃഷ്ടിച്ചെന്ന് കണ്ടെത്തുന്നതിൽ വിദഗ്ധം. ഇൻസ്റ്റോൾ ചെയ്ത ആപ്പുകളുമായി ഫയലുകൾ താരതമ്യം ചെയ്ത്, സ്‌പേസ് സുരക്ഷിതമായി ഒഴിപ്പിക്കാൻ ഓപ്ഷനുകൾ നൽകുന്നു.

✨ അൺഇൻസ്റ്റോൾ ചെയ്ത ശേഷം ക്ലീൻ ചെയ്യുക

ആപ്പുകൾ നിർദ്ദിഷ്ട ഫോൾഡറിന് പുറത്ത് ഫയലുകൾ സൃഷ്ടിക്കുകയാണെങ്കിൽ, അൺഇൻസ്റ്റോൾ ചെയ്ത ശേഷവും ഫയലുകൾ നിലനിൽക്കും. "CorpseFinder" ടൂൾ ആപ്പ് അവശിഷ്ടങ്ങൾ കണ്ടെത്തുന്നു, ഏത് ആപ്പിന്റേതാണെന്ന് പറഞ്ഞ് ഡിലീറ്റ് ചെയ്യാൻ സഹായിക്കുന്നു.

🔍 ഉപകരണം സ്മാർട്ടായി തിരയുക

ശൂന്യ ഫോൾഡറുകൾ, താൽക്കാലിക ഫയലുകൾ, ഇൻസ്റ്റോൾ ചെയ്ത ആപ്പുകൾ എന്നിവ ഫിൽട്ടർ ചെയ്യുക. സ്വന്തം തിരയൽ മാനദണ്ഡങ്ങൾ ഉണ്ടാക്കാം. "SystemCleaner" ടൂൾ വ്യത്യസ്ത മാനദണ്ഡങ്ങളിൽ ഫയലുകൾ സ്വയം തിരഞ്ഞ് ഇല്ലാതാക്കുന്നു.

🧹 അനാവശ്യ ഫയലുകളും കാഷെകളും ഇല്ലാതാക്കുക

തംബ്‌നെയിലുകൾ, ട്രാഷ്, ഓഫ്‌ലൈൻ കാഷെകൾ: ആപ്പുകൾ ക്ലീൻ ചെയ്യുന്നില്ലെങ്കിൽ "AppCleaner" ചെയ്യും.

📦 എല്ലാ ആപ്പുകളും മാനേജ് ചെയ്യുക

ഇൻസ്റ്റോൾ ചെയ്ത ആപ്പുകളുടെ സമഗ്ര ലിസ്റ്റ്. പ്രവർത്തനക്ഷമം, പ്രവർത്തനരഹിതം, ഉപയോക്താവ് അല്ലെങ്കിൽ സിസ്റ്റം ആപ്പ് — ഒരു ആപ്പും ഒളിക്കില്ല. "AppControl" ടൂൾ ആപ്പുകൾ തിരയാനും, അടുക്കാനും, നിയന്ത്രിക്കാനും അനുവദിക്കുന്നു.

📊 നിങ്ങളുടെ സ്‌പേസ് ഉപയോഗിക്കുന്നത് എന്ത്

ഫോൺ സ്റ്റോറേജ്, SD കാർഡ്, USB ഉപകരണങ്ങളിലൂടെ ആപ്പ്, മീഡിയ, സിസ്റ്റം ഫയലുകൾ നിര്‍വ്വഹിക്കൽ സങ്കീർണ്ണമാണ്. "StorageAnalyzer" ഉപകരണത്തിൽ സ്‌പേസ് എങ്ങനെ ഉപയോഗിക്കുന്നുവെന്ന് കാണിക്കുന്നു.

📷 ഡ്യൂപ്ലിക്കേറ്റ് ഡേറ്റ കണ്ടെത്തുക

ഡ്യൂപ്ലിക്കേറ്റ് ഡൗൺലോഡുകൾ, സോഷ്യൽ മീഡിയ ഫോട്ടോകൾ, സമാന ചിത്രങ്ങൾ — കോപ്പികൾ അടിഞ്ഞുകൂടാം. "Deduplicator" ടൂൾ ഒരേ അല്ലെങ്കിൽ സമാന ഫയലുകൾ കണ്ടെത്തി അധിക കോപ്പികൾ ഇല്ലാതാക്കാൻ സഹായിക്കുന്നു.

👆 ഒരു സ്വൈപ്പ് ഉപയോഗിച്ച് അടുക്കൊഴിക്കുക

ഒരു ഫോൾഡർ തിരഞ്ഞെടുക്കുക, ഫയലുകൾ ഒന്നൊന്നായി സ്വൈപ്പ് ചെയ്ത് സൂക്ഷിക്കാനോ, ഇല്ലാതാക്കാനോ, ഒഴിവാക്കാനോ ചെയ്യുക. "Swiper" ടൂൾ ഇല്ലാതാകുന്നതിന് മുൻപ് തീരുമാനങ്ങൾ അവലോകനം ചെയ്യാൻ സഹായിക്കുന്നു; സെഷൻ പിന്നീടും തുടരാം.

🗜️ ഫോട്ടോകൾ ഇല്ലാതാക്കാതെ സ്ഥലം ലാഭിക്കുക

ഫോട്ടോകൾ ധാരാളം സ്റ്റോറേജ് ഉപയോഗിക്കും, പക്ഷേ നീക്കം ചെയ്യാൻ ആഗ്രഹിക്കില്ലായിരിക്കും. "Media Squeeze" ടൂൾ ക്രമീകരിക്കാവുന്ന ഗുണനിലവാര ക്രമീകരണങ്ങളോടെ ചിത്രങ്ങൾ കംപ്രസ് ചെയ്ത് സ്ഥലം ലഭ്യമാക്കുന്നു.

ഈ ആപ്പ് പരസ്യരഹിതമാണ്. ചില സവിശേഷതകൾക്ക് പണമടച്ചുള്ള നവീകരണം ആവശ്യമാണ്.

SD Maid 2/SE SD Maid 1/Legacy-യുടെ പിൻഗാമിയാണ്.
പുതിയ Android പതിപ്പുകൾക്കായി ഒപ്റ്റിമൈസ് ചെയ്ത്, വൃത്തിയാക്കുന്നതിൽ ശ്രദ്ധ കേന്ദ്രീകരിച്ചു.

ഈ ആപ്പിന് ഐച്ഛിക സവിശേഷതകളുണ്ട്, അവ AccessibilityService API ഉപയോഗിച്ച് ആവർത്തിത്വ പ്രവൃത്തികൾ ഓട്ടോമേറ്റ് ചെയ്യുന്നു.
AccessibilityService API ഉപയോഗിച്ച്, ഒന്നിലധികം ആപ്പുകളിൽ ബട്ടണുകൾ ക്ലിക്ക് ചെയ്ത് പ്രവർത്തനങ്ങൾ നടത്താൻ കഴിയും, ഉദാ: കാഷെ ഡിലീറ്റ് ചെയ്യൽ.
ഈ ആപ്പ് വിവരശേഖരണത്തിനായി AccessibilityService API ഉപയോഗിക്കുന്നില്ല.
