<i>SD Maid 2/SE</i> നിങ്ങളുടെ Android-ന്റെ വിശ്വസ്ത സഹായിയാണ്, അത് വൃത്തിയായും ചിട്ടയായും സൂക്ഷിക്കാൻ.

ആരും തികഞ്ഞവരല്ല, Android-ഉം ഇല്ല.

* നിങ്ങൾ നീക്കംചെയ്‌ത ആപ്പുകൾ എന്തെങ്കിലും ഫയലുകൾ അവശേഷിപ്പിക്കുന്നു.
* ലോഗുകൾ, ക്രാഷ് റിപ്പോർട്ടുകൾ, നിങ്ങൾക്ക് വേണ്ടാത്ത മറ്റ് ഫയലുകൾ നിരന്തരം സൃഷ്ടിക്കപ്പെടുന്നു.
* നിങ്ങളുടെ സ്റ്റോറേജ് നിങ്ങൾക്ക് പരിചിതമല്ലാത്ത ഫയലുകളും ഫോൾഡറുകളും ശേഖരിക്കുകയാണ്.
* നിങ്ങളുടെ ഗ്യാലറിയിൽ ഡ്യൂപ്ലിക്കേറ്റ് ഫോട്ടോകൾ.

ഇനി ഇവിടെ തുടരേണ്ട… SD Maid 2/SE നിങ്ങളെ സഹായിക്കട്ടെ!

SD Maid 2/SE ഒരു ആപ്പ്, ഫയൽ മാനേജരാണ്, ഏതൊക്കെ ആപ്പ് ഏതൊക്കെ ഫയൽ സൃഷ്ടിച്ചെന്ന് കണ്ടെത്തുന്നതിൽ വിദഗ്ധം. ഇൻസ്റ്റോൾ ചെയ്ത ആപ്പുകളുമായി ഫയലുകൾ താരതമ്യം ചെയ്ത്, സ്‌പേസ് സുരക്ഷിതമായി ഒഴിപ്പിക്കാൻ ഓപ്ഷനുകൾ നൽകുന്നു.

✨ ആപ്പുകൾ അൺഇൻസ്റ്റോൾ ചെയ്ത ശേഷം ക്ലീൻ അപ്പ് ചെയ്യുക

ആപ്പുകൾ അവയ്ക്ക് നിർദ്ദേശിച്ച ഫോൾഡറുകൾക്ക് പുറത്ത് ഫയലുകൾ സൃഷ്ടിക്കുകയാണെങ്കിൽ, ആ ആപ്പുകൾ അൺഇൻസ്റ്റാൾ ചെയ്ത ശേഷവും ഫയലുകൾ നിലനിൽക്കാം. "CorpseFinder" ടൂൾ ആപ്പ് അവശിഷ്ടങ്ങൾ കണ്ടെത്തുന്നു, അവ ഏത് ആപ്പിന്റേതാണെന്ന് നിങ്ങളോട് പറഞ്ഞ് അവ ഡിലീറ്റ് ചെയ്യാൻ സഹായിക്കുന്നു.

🔍 സ്മാർട്ടായ രീതിയിൽ നിങ്ങളുടെ ഉപകരണം തിരയുക

ശൂന്യമായ ഫോൾഡറുകൾ, താൽക്കാലിക ഫയലുകൾ, ഇതിനകം ഇൻസ്റ്റാൾ ചെയ്ത ആപ്പുകൾ എന്നിവയ്ക്കും മറ്റും ഫിൽട്ടർ ചെയ്യുക. നിങ്ങൾക്ക് സ്വന്തം തിരയൽ മാനദണ്ഡങ്ങളും സൃഷ്ടിക്കാൻ കഴിയും. "SystemCleaner" ടൂൾ വ്യത്യസ്ത മാനദണ്ഡങ്ങളിൽ ഫയലുകൾ സ്വയം തിരഞ്ഞ് ഇല്ലാതാക്കുന്നു.

🧹 അനാവശ്യ ഫയലുകളും മറഞ്ഞിരിക്കുന്ന കാഷെകളും ഇല്ലാതാക്കുക

തംബ്‌നെയിലുകൾ, ട്രാഷ് ബിൻ, ഓഫ്‌ലൈൻ കാഷെകൾ, മറ്റും: ആപ്പുകൾ സ്വയം ക്ലീൻ ചെയ്യുന്നില്ലെങ്കിൽ, ഈ ആപ്പ് ചെയ്യും. "AppCleaner" ടൂൾ അനാവശ്യ ഫയലുകളുള്ള ആപ്പുകൾ കണ്ടെത്തുന്നു.

📦 നിങ്ങളുടെ എല്ലാ ആപ്പുകളും മാനേജ് ചെയ്യുക

നിങ്ങളുടെ ഉപകരണത്തിൽ ഇൻസ്റ്റോൾ ചെയ്ത എല്ലാ ആപ്പുകളുടെയും സമഗ്രമായ ഒരു ലിസ്റ്റ് നേടുക. പ്രവർത്തനക്ഷമം, പ്രവർത്തനരഹിതം, ഉപയോക്താവ് അല്ലെങ്കിൽ സിസ്റ്റം ആപ്പ്: ഒരു ആപ്പിനും നിങ്ങളിൽ നിന്ന് ഒളിക്കാൻ കഴിയില്ല. "AppControl" ടൂൾ ആപ്പുകൾ തിരയാനും, അടുക്കാനും, നിയന്ത്രിക്കാനും അനുവദിക്കുന്നു.

📊 നിങ്ങളുടെ സ്‌പേസ് ഉപയോഗിക്കുന്നത് എന്താണ്

ഡിവൈസ് സ്റ്റോറേജ്, SD കാർഡുകൾ, USB ഉപകരണങ്ങൾ എന്നിവയിലുടനീളം ആപ്പുകൾ, മീഡിയ, സിസ്റ്റം, മറ്റ ഫയലുകൾ എന്നിവ ഉള്ളതിനാൽ സ്റ്റോറേജ് മാനേജ്‌മെന്റ് സങ്കീർണ്ണമായിരിക്കും. "StorageAnalyzer" ഒരു ഫയൽ മാനേജരാണ്, അത് നിങ്ങളുടെ ഉപകരണത്തിൽ സ്‌പേസ് എങ്ങനെ ഉപയോഗിക്കുന്നുവെന്ന് പ്രദർശിപ്പിക്കുന്നു, നിങ്ങളുടെ സ്റ്റോറേജ് മാനേജ്‌മെന്റ് ലളിതമാക്കുന്നു.

📷 ഡ്യൂപ്ലിക്കേറ്റ് ഡേറ്റ കണ്ടെത്തുക

ഡ്യൂപ്ലിക്കേറ്റ് ഡൗൺലോഡുകൾ, സോഷ്യൽ മീഡിയ വഴി അയച്ച ഫോട്ടോകൾ അല്ലെങ്കിൽ ഒരേ രംഗത്തിന്റെ സമാന ചിത്രങ്ങൾ: കാലക്രമേണ കോപ്പികൾ അടിഞ്ഞുകൂടാം. "Deduplicator" ടൂൾ കൃത്യമായോ സമാനമായോ ഉള്ള ഫയലുകൾ കണ്ടെത്തി അധിക കോപ്പികൾ ഇല്ലാതാക്കാൻ സഹായിക്കുന്നു.

👆 ഒരു സ്വൈപ്പ് ഉപയോഗിച്ച് അടുക്കൊഴിക്കുക

ഒരു ഫോൾഡർ തിരഞ്ഞെടുക്കുക, ലളിതമായ സ്വൈപ്പുകൾ ഉപയോഗിച്ച് ഫയലുകൾ ഒന്നൊന്നായി സൂക്ഷിക്കാനോ, ഇല്ലാതാക്കാനോ, ഒഴിവാക്കാനോ കടന്നുപോകുക. "Swiper" ടൂൾ ഉപയോഗിച്ച് എന്തെങ്കിലും ഇല്ലാതാകുന്നതിന് മുൻപ് എല്ലാ തീരുമാനങ്ങളും അവലോകനം ചെയ്യാം, കൂടാതെ ഒരു സെഷൻ സംരക്ഷിച്ച് പിന്നീട് തുടരാം.

🗄️ ഫോട്ടോകൾ ഇല്ലാതാക്കാതെ സ്ഥലം ലാഭിക്കുക

ഫോട്ടോകൾ ധാരാളം സ്റ്റോറേജ് ഉപയോഗിക്കും, പക്ഷേ നിങ്ങൾ എല്ലായ്‌പ്പോഴും അവ നീക്കം ചെയ്യാൻ ആഗ്രഹിക്കുന്നില്ല. "Media Squeeze" ടൂൾ ക്രമീകരിക്കാവുന്ന ഗുണനിലവാര ക്രമീകരണങ്ങളോടെ നിങ്ങളുടെ ചിത്രങ്ങൾ കംപ്രസ് ചെയ്ത് ഫോട്ടോകൾ സൂക്ഷിച്ചുകൊണ്ട് സ്ഥലം ലഭ്യമാക്കുന്നു.

ഈ ആപ്പ് പരസ്യരഹിതമാണ്. ചില സവിശേഷതകൾക്ക് പണമടച്ചുള്ള നവീകരണം ആവശ്യമാണ്.

SD മെയ്ഡ് 2/SE എന്നത് SD മെയ്ഡ് 1/ലെഗസിയുടെ പിൻഗാമിയാണ്.
പുതിയ Android പതിപ്പുകൾക്കായി ഒപ്റ്റിമൈസ് ചെയ്‌ത് വൃത്തിയാക്കുന്നതിൽ ശ്രദ്ധ കേന്ദ്രീകരിച്ചു.

ഈ ആപ്പിന് ഐച്ഛിക സവിശേഷതകളുണ്ട്, അവ മടുപ്പിക്കുന്ന പ്രവർത്തനങ്ങൾ യാന്ത്രികമാക്കാൻ AccessibilityService API ഉപയോഗിക്കുന്നു.
AccessibilityService API ഉപയോഗിച്ച്, ഈ ആപ്പിന് നിങ്ങൾക്ക് വേണ്ടി ബട്ടണുകൾ ക്ലിക്ക് ചെയ്ത് ഒന്നിലധികം ആപ്പുകളിൽ പ്രവർത്തനങ്ങൾ നടത്താൻ കഴിയും, ഉദാ: deleting caches.
ഈ ആപ്പ് വിവരശേഖരണത്തിനായി AccessibilityService API ഉപയോഗിക്കുന്നില്ല.

SD Maid 2/SE ഒരു ഫയൽ മാനേജർ & ക്ലീനർ ആപ്പ് ആണ്.
