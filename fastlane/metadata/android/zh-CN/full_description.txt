<i>SD Maid SE</i>是一款值得信赖的安卓手机清理工具，可以帮助你保持手机的整洁。

人无完人，Android 亦是如此。
* 你已卸载的应用会留下一些残留。
* 各种日志、崩溃报告以及其他你不需要的文件会不断地被创建出来。
* 你的存储空间在收集一些你不认识的文件和目录。

不用多说... 让 SD Maid SE 帮助你：

* 清理已卸载应用的数据
* 查找隐藏的应用缓存。
* 清理多余的系统文件

SD Maid SE 没有广告。 有些功能需要付费升级才能使用。

SD Maid 2/SE 是 SD Maid 1/Legacy 的更新版本。
对新版本 Android 进行了优化，并专注于清理。

SD Maid SE 拥有可选功能，利用无障碍服务 API 自动执行繁琐的动作。
使用无障碍服务 API，SD Maid SE 可以为您点击按钮，执行多个应用程序的操作，例如： 清除缓存。
SD Maid SE 不会使用无障碍服务 API 来收集信息。
