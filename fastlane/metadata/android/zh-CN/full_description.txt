<i>SD Maid SE</i> 是您的 Android 手机的可靠助手，帮助您保持手机清洁整洁。

人无完人，Android 亦是如此。
* 卸载的应用会留下残留。
* 会不断地创建各种日志、崩溃报告以及其他您不需要的文件。
* 存储空间中会积累一些您不认识的文件和目录。

让我们不再浪费时间了… 让 SD Maid SE 帮助您保持手机清净整洁吧：

* 清理已卸载应用的数据
* 查找隐藏的应用缓存
* 清理多余的系统文件

SD Maid SE 没有广告。 部分功能需要付费升级才能使用。

SD Maid 2/SE 是 SD Maid 1/Legacy 的更新版本。
对新版本 Android 进行了优化，并专注于清理。

SD Maid SE 拥有可选功能，可利用无障碍服务 API 自动执行繁琐的动作。
通过无障碍服务 API，SD Maid SE 可以替您点击按钮来执行多中应用程序的操作，例如： 清除缓存。
SD Maid SE 不会使用无障碍服务 API 来收集信息。
