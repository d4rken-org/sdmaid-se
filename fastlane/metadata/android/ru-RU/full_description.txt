<i>SD Maid SE</i> - надежный помощник Вашего Android, обеспечит ему чистоту и порядок.

Никто не идеален, и Android тоже.
* Приложения, которые Вы удалили, оставляют что-то после себя.
* Журналы, отчеты о сбоях и другие файлы, которые Вам на самом деле не нужны, постоянно создаются.
* В памяти Вашего устройства скапливаются неизвестные Вам файлы и папки.

Поехали дальше… Позвольте SD Maid SE Вам помочь:

* Очистить данные уже удаленных приложений
* Найти скрытый кэш приложений
* Удалить лишние системные файлы

В SD Maid SE нет рекламы. Некоторые функции требуют платного обновления.

SD Maid 2/SE является наследником SD Maid 1/(устаревшего приложения).
Оптимизировано для новых версий Android и сфокусировано на очистке.

SD Maid SE имеет дополнительные функции, использующие службу специальных возможностей для автоматизации утомительных действий.
Используя службу специальных возможностей, SD Maid SE может вместо Вас нажимать кнопки для выполнения операций с несколькими приложениями, например, при удалении кэша.
SD Maid SE не использует службу специальных возможностей для сбора информации.
