<i>SD Maid 2/SE</i> - надежный помощник Вашего Android, обеспечит ему чистоту и порядок.

Никто не идеален, и Android тоже.

* Приложения, которые Вы удалили, оставляют что-то после себя.
* Журналы, отчеты о сбоях и другие файлы, которые Вам не нужны, постоянно создаются.
* В памяти Вашего устройства скапливаются неизвестные Вам файлы и папки.
* Дубликаты фотографий в Вашей галерее.

Довольно… Позвольте SD Maid 2/SE помочь Вам!

SD Maid 2/SE — это менеджер приложений и файлов, который знает, какие приложения создали файлы на Вашем устройстве. Он сравнивает файлы с установленными приложениями, помогая безопасно освободить место в хранилище.

✨ Очистка после удаления приложений

Если приложения создают файлы за пределами назначенных папок, то файлы могут остаться даже после удаления приложений. Инструмент "CorpseFinder" находит остатки, рассказывает Вам, к какому приложению они принадлежали, и помогает удалить их.

🔍 Умный поиск на устройстве

Фильтр по пустым папкам, временным файлам, уже установленным приложениям и многое другое. Вы даже можете создать свой собственный критерий поиска. Инструмент «Система» автоматически ищет и удаляет файлы по различным критериям.

🧹 Удаление ненужных файлов и скрытого кэша

Эскизы, корзины, оффлайн-кэш и многое другое: если приложения не очищают сами себя, это сделает данное приложение. Инструмент "AppCleaner" находит приложения с излишними файлами.

📦 Управляйте всеми Вашими приложениями

Получите полный список всех приложений, установленных на Вашем устройстве. Работающее, не работающее, пользовательское или системное: ни одно приложение не может скрыться от Вас. Инструмент «Менеджер» — это менеджер приложений для поиска, сортировки и управления Вашими приложениями.

📊 Что использует Вашу память

Управлять памятью может быть сложно: приложения, медиа, системные и другие файлы расположены в памяти устройства, на SD-картах и USB-накопителях. "StorageAnalyzer" - это файловый менеджер, который отображает, как используется хранилище на Вашем устройстве, упрощая управление памятью.

📷 Поиск дубликатов данных

Дублирующиеся загрузки, фотографии, отправленные через социальные сети или просто одинаковые фото в том же месте: со временем копии могут накапливаться. Инструмент "Deduplicator" находит файлы, которые являются одинаковыми или похожими и помогает удалять лишние копии.

👆 Очистка с помощью свайпа

Выберите папку и переходите по файлам один за одним простыми свайпами — оставляйте, удаляйте или пропускайте. Инструмент «Свайпер» позволяет просмотреть все решения перед удалением и сохранить сеанс, чтобы продолжить позже.

🗄️ Экономьте место без удаления фото

Фотографии могут занимать много места, но Вы не всегда хотите их удалять. Инструмент «Сжатие медиа» сжимает изображения с настраиваемым качеством, чтобы освободить место, сохраняя Ваши фотографии.

Это приложение не содержит рекламы. Некоторые функции требуют платного обновления.

SD Maid 2/SE является наследником SD Maid 1/(устаревшего приложения).
Оптимизировано для новых версий Android и сфокусировано на очистке.

В приложении имеются дополнительные функции, использующие службу специальных возможностей для автоматизации действий.
С помощью API AccessibilityService это приложение может активировать для Вас нажатие кнопок для выполнения операций с несколькими приложениями, например, при удалении кэша.
Это приложение не использует API AccessibilityService для сбора информации.

SD Maid 2/SE - это файловый менеджер и приложение для очистки.
