<i>SD Maid SE</i> - надійний помічник вашого Android, щоб тримати його в чистоті та порядку.

Ніхто не є ідеальним, і Android теж.
* Програми, які ви вже видалили, залишають щось після себе.
* Постійно створюються логи, звіти про збої та інші файли, які вам не потрібні.
* У пам'яті вашого телефону накопичуються файли та каталоги, які ви не розпізнаєте.

Не будемо продовжувати... Дозвольте SD Maid SE допомогти вам:

* Очистити дані з вже видалених програм
* Знайти прихований кеш програм
* Видалити зайві системні файли

SD Maid SE не містить жодної реклами. Деякі функції потребують платного апґрейда.

SD Maid 2/SE є наступником SD Maid 1/Legacy.
Оптимізований для нових версій Android і зосереджений на очищенні.

SD Maid SE має додаткові функції, які використовують AccessibilityService API для автоматизації нудних дій.
Використовуючи AccessibilityService API, SD Maid SE може натискати кнопки для виконання операцій у декількох додатках, наприклад, видалення кешу.
SD Maid SE не використовує AccessibilityService API для збору інформації.
