<i>SD 女僕 SE</i> 是您 Android 的值得信賴的小幫手，她能帶給您乾淨與整潔。

沒有人是完美的，Android 也是如此。
* 您已經移除的應用程式會留下一些東西。
* 記錄檔、當機報告和其他您不想要的檔案被不斷建立，
* 您的儲存空間正在收集您無法辨識的檔案和目錄。

不要止步於此… SD 女僕 SE 可以幫您：

* 清理來自已經解除安裝的應用程式的資料
* 尋找隱藏的應用程式快取
* 移除非必要的系統檔案

SD 女僕 SE 是無廣告的。 部分功能需要付費升級。

SD 女僕 2/SE 是 SD 女僕 1/舊版的後繼者。
專為更新的 Android 版本最佳化，並專注於清理工作。

SD 女僕 SE 有一些選用功能，利用無障礙服務 API 自動完成乏味的動作。
使用無障礙服務 API，SD 女僕 SE 可以為您按下按鈕，並對多個應用程式進行處理作業，例如： 正在刪除快取。
SD 女僕 SE 不會使用無障礙服務 API 收集資訊。
