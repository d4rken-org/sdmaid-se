<i>SD Maid 2/SE</i> es el asistente de confianza de tu Android, para mantenerlo limpio y ordenado.

Nadie es perfecto y Android tampoco lo es.

* Las aplicaciones desinstaladas dejan restos.
* Se están creando constantemente registros, informes de fallos y otros archivos que no deseas.
* Tu almacenamiento está recopilando archivos y carpetas que no reconoces.
* Fotos duplicadas en tu galería.

¡Deja que SD Maid 2/SE te ayude!

SD Maid 2/SE es una aplicación y gestor de archivos que se especializa en saber qué aplicaciones han creado archivos específicos en tu dispositivo. SD Maid 2/SE busca en tu dispositivo y compara los archivos con las aplicaciones instaladas para ofrecerte opciones para liberar espacio de almacenamiento de forma segura.

✨ Limpia después de desinstalar las aplicaciones

Si las aplicaciones crean archivos fuera de sus carpetas designadas, entonces los archivos pueden permanecer incluso después de desinstalar las aplicaciones. La herramienta "Buscador de restos" encuentra restos de aplicaciones, te dice a qué aplicación pertenecían y te ayuda a eliminarlos.

🔍 Busca en tu dispositivo de forma inteligente

Filtra las carpetas vacías, los archivos temporales, las aplicaciones ya instaladas y mucho más. Incluso puede crear sus propios criterios de búsqueda. La herramienta "Limpiador del Sistema" te permite buscar automáticamente en tu dispositivo y eliminar archivos en función de varios criterios.

🧹 Eliminar archivos prescindibles y cachés ocultos

Miniaturas, papeleras, cachés sin conexión y mucho más: Si las aplicaciones no se limpian solas, esta aplicación lo hará. La herramienta "Limpiador de Aplicaciones" encuentra aplicaciones con archivos prescindibles.

📦 Gestiona todas tus aplicaciones

Obtén una lista completa de todas las aplicaciones instaladas en tu dispositivo. Aplicación activada, desactivada, de usuario o de sistema: ninguna aplicación puede esconderse de ti. La herramienta "Control de Aplicaciones" es un gestor de aplicaciones que te permite buscar, ordenar y controlar tus aplicaciones.

📊 ¿Qué es lo que ocupa todo tu espacio?

La gestión del almacenamiento puede ser compleja, con aplicaciones, medios, sistema y otros archivos en el almacenamiento del teléfono, tarjetas SD y dispositivos USB. "Analizador de Almacenamiento" es un gestor de archivos que muestra cómo se utiliza el espacio en tu dispositivo, simplificando la gestión de tu almacenamiento.

📷  Encontrar datos duplicados

Descargas duplicadas, fotos enviadas a través de las redes sociales o simplemente fotos similares de la misma escena: Con el tiempo pueden acumularse copias. La herramienta "Eliminador de Duplicados" encuentra archivos exactamente iguales o similares y te ayuda a eliminar las copias sobrantes.

👆 Desordena con un deslizamiento

Elige una carpeta y revisa los archivos uno a uno con simples deslizamientos para conservar, eliminar u omitir. La herramienta «Swiper» te permite revisar todas las decisiones antes de eliminar nada, y puedes guardar una sesión para continuar más tarde.

🗄️ Ahorra espacio sin eliminar fotos

Las fotos pueden ocupar mucho almacenamiento, pero no siempre quieres eliminarlas. La herramienta «Media Squeeze» comprime tus imágenes con ajustes de calidad configurables para liberar espacio sin perder tus fotos.

Esta aplicación no tiene publicidad. Algunas funciones requieren la versión de pago.

SD Maid 2/SE es el sucesor de SD Maid 1/Legacy.
Optimizado para nuevas versiones de Android y enfocado en la limpieza.

Esta aplicación tiene funciones opcionales que utilizan la API del Servicio de Accesibilidad para automatizar acciones tediosas.
Usando la API AccessibilityService, esta aplicación puede hacer clic en botones para que pueda realizar operaciones en varias aplicaciones, por ejemplo. borrando la memoria caché.
Esta aplicación no utiliza la API del Servicio de Accesibilidad para recopilar información.

SD Maid 2/SE es una aplicación de limpieza y gestión de archivos.
