<i>SD Maid SE</i> es el asistente de confianza para tu Android, para mantenerlo limpio y ordenado.

Nadie es perfecto y Android tampoco lo es.
* Las aplicaciones desinstaladas dejan restos.
* Registros, informes de fallos y otros archivos que realmente no quieres se crean constantemente.
* Tu almacenamiento está acumulando archivos y directorios que no reconoces.

No sigamos aquí... Deje que SD Maid SE le ayude:

* Limpia datos de aplicaciones ya desinstaladas
* Borrar los archivos innecesarios y las cachés ocultas
* Eliminar los archivos del sistema que no se usan
* Crea tus propios filtros para archivos aleatorios
* Encontrar y eliminar los datos duplicados
* Analiza el almacenamiento y elimina los archivos grandes que ocupan espacio

SD Maid SE no tiene publicidad. Algunas funciones requieren la versión de pago.

SD Maid 2/SE es el sucesor de SD Maid 1/Legacy.
Optimizado para nuevas versiones de Android y enfocado en la limpieza.

SD Maid SE tiene funciones opcionales que utilizan la API AccessibilityService para automatizar las tareas tediosas.
SD Maid SE puede presionar botones para que realice operaciones en múltiples aplicaciones utilizando la API AccessibilityService, como por ejemplo. borrando la memoria caché.
SD Maid SE no utiliza el servicio de accesibilidad para recopilar información.
